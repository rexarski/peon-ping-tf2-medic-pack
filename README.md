# TF2 Medic Sound Pack

A [CESP](https://openpeon.com/spec) (Coding Event Sound Pack) featuring voice lines from the Medic class in Team Fortress 2.

## Installation

Copy this pack to your CESP packs directory:

```bash
cp -r tf2-medic-pack ~/.openpeon/packs/tf2-medic
```

Or symlink it:

```bash
ln -s /path/to/tf2-medic-pack ~/.openpeon/packs/tf2-medic
```
